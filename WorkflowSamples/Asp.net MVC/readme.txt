This sample demos how to upload/tranlsate model files from asp.net mvc application.

To use:
Build the .NET sample with Client code, secret and bucket name

.NET sample can
               1.            Create token
               2.            upload files, 
               3.            Translate files 
               4.            Generate urn.

Now use mongoose to host the MyHTML.htm file. This htm sample takes a token & urn as input. MyHTML.htm, then shows the file in viewer.
